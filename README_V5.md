# ガチャドコヤ v5

## 今回の追加

- 商品詳細ページ `/products/[id]`
- 店舗詳細ページ `/locations/[id]`
- 目撃詳細の商品名・店舗名から各詳細ページへ移動
- 「助かった！」ボタン（まずは端末内に保存）
- 商品保存・店舗保存
- 商品ごとの目撃場所一覧
- 店舗ごとの設置ガチャ一覧
- 将来「助かった！」を全ユーザー共有にするSQL `supabase/schema-v5.sql`

## 更新方法

`gachadokoya-v5` フォルダの中身だけを GitHub の `gachadokoya-v2` にアップロードし、必ず `Commit changes` を押してください。

今回は `schema-v5.sql` を実行しなくてもページは動きます。「助かった！」は現時点ではブラウザ端末内に保存されます。
